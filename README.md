# Cancer_Type_Detection_using_image_and_report_generation
This project uses deep learning and image processing to detect potential cancer from medical images. A trained CNN model analyzes image patterns to assist early diagnosis. Built using Python, TensorFlow/Keras, and OpenCV, the system demonstrates how AI can support faster and more accurate medical image analysis.
The system processes medical images, extracts important visual features, and predicts whether the image indicates a cancerous or non-cancerous condition. This demonstrates how Artificial Intelligence can support healthcare diagnostics by improving speed and accuracy in image-based analysis.
